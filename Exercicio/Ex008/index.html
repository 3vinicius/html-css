Principais Formatações

Negrito / Destaque

Nessa frase usamos o <b> colocar em negrito
Podemos substituir por <strong> que tem sentido

Italico / ênfase

Para ultilizar italico ultilizamos o <i>
Para dar sentido ultilizamos o <em>

Texto marcado

Para fazer uma marcação em um texto ultilizamos <mark>
